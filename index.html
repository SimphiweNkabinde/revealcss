Add graceful animations to your web pages

A lightweight CSS library for CSS animations.

reveal.css allows you to easily add simple but graceful animations to your project by using simple classnames on the elements you want animated

Usage

Adding any of the reveal.css animation classes to an element tag will apply animation styling to that element.

Fade

The fade animation includes fade-in and fade-out classnames with modifiers for specifying the animation's direction.

Notation: fade-{in/out}-{left/right/top/bottom}

Examples

<h1 class="fade-in">fade in</h1>
<h1 class="fade-in-left">fade in left</h1>
<h1 class="fade-out-right">fade out right</h1>

Flip

The flip animation includes flip-in and flip-out classnames with modifiers for specifying the animation's direction.

Notation: flip-{in/out}-{x/y}

Examples

<h1 class="flip-in-x">flip in x</h1>
<h1 class="flip-in-y">flip in y</h1>
<h1 class="flip-out-x">flip out x</h1>
