Slide Simple

A light-weight CSS and JavaScript library for simple transition animations.

These css classes will cause the animation to occur as the page loads

class="ss-top"

Speed

slow normal fast

animation Direction

To invoke slide animations when the relevant element is scrolled on, use the following classes

ss-onscroll-*
*-right
*-left-slow
*-right-fast
*-top
*-bottom-slow
*-top-fast
*-center
*-center-fast
